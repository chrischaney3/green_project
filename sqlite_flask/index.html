Flask Powered API

The routes for our Flask App include a JSON representation of the different fueling station types and station names, a query of the cities in each state that contain electric vehicle charging stations, a count of all the charging stations in each state, and a count of the accessibility of each station. The accessibility query determined that there are many more public charging stations than private charging stations in the United States.

Here are the available routes:

/fuelingStationTypes

/name

/cities

/count

/accessibility
